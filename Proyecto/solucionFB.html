EXPLICACIÓN DEL PROBLEMA
Un prefijo se define como un elemento gramatical que se adhiere a un término para cambiar su significado, es decir que, se anteponen a la palabra que se desea modificar.
Entonces, dado un conjunto de cadenas (4) se encontrará la cadena de prefijo común más larga, que es el prefijo de todas las cadenas dadas.

Instrucciones

Inserte las 4 cadenas a comparar y presione el botón comenzar.

NOTA: Las palabras tienen que estar en minusculas.











Explicación del Algoritmo

PASO 1

Se seleccionan 2 cadenas para que asi se empiece a comparar caracter por caracter,de inicio a fin.

PASO 2

Dichos caracteres serán almacenados en la variable prefix, la cual nos indica el prefijo común más largo hasta el momento.

PASO 3

Si se obtuvo un prefijo con las cadenas utilizadas, dicho prefijo se comparará con la siguiente cadena.
Finalmente, se vuelve a repetir el mismo proceso con las cadenas faltantes, que para este ejemplo son 4.


prefijoComunDos

function prefijoComunDos(str1,str2)

{

   let result = "";

   let n1 = str1.length, n2 = str2.length;

   for (let i = 0, j = 0; i <= (n1 - 1) && j <=(n2 - 1); i++, j++) {

     if (str1[i] != str2[j])

       break;

     result += str1[i];

    }

return (result);

}

prefijoComunMasLargo

function prefijoComunMasLargo(arr,n)

{

     let prefix = arr[0];

     for (let i = 1; i <= n - 1; i++)

       prefix = prefijoComunDos(prefix, arr[i]);

     return (prefix);

}

Análisis de la Cota

El análisis de la cota se realizó teniendo en cuenta el peor caso del algoritmo, que se da cuando todas las palabras que se encuentran en el String para obtener su prefijo común más largo es la misma palabra. De esta manera la cota es O(n1*n), en donde “n1” representa el número de caracteres que tiene la palabra y “n” representa el número de palabras contenidas en el String para sacar su prefijo común más largo.
